## 2
–ù–µ—Ö–∞–π $a$, $b$, $c$ ‚Äì —Å—Ç–æ—Ä–æ–Ω–∏ –¥–µ—è–∫–æ–≥–æ —Ç—Ä–∏–∫—É—Ç–Ω–∏–∫–∞. –ó–Ω–∞–π—Ç–∏ –Ω–∞–π–º–µ–Ω—à–µ –∑–Ω–∞—á–µ–Ω–Ω—è $k$, –¥–ª—è —è–∫–æ–≥–æ –∑–∞–≤–∂–¥–∏ –≤–∏–∫–æ–Ω—É—î—Ç—å—Å—è –Ω–µ—Ä—ñ–≤–Ω—ñ—Å—Ç—å $\left| \frac{a-b}{a+b} + \frac{b-c}{b+c} + \frac{c-a}{c+a}\right| < k$.
<details><summary>–í—ñ–¥–ø–æ–≤—ñ–¥—å</summary>

$k = \frac{8\sqrt{2} - 5\sqrt{5}}{3}$
<details><summary>–†–æ–∑–≤‚Äô—è–∑–∞–Ω–Ω—è</summary>

–®–ª—è—Ö–æ–º –ø—Ä–æ—Å—Ç–∏—Ö –ø–µ—Ä–µ—Ç–≤–æ—Ä–µ–Ω—å –º–æ–∂–µ–º–æ –æ–¥–µ—Ä–∂–∞—Ç–∏ —Ä—ñ–≤–Ω—ñ—Å—Ç—å: $\frac{a-b}{a+b}+ \frac{b-c}{b+c}+\frac{c-a}{c+a} = \frac{a-b}{a+b}\cdot \frac{b-c}{b+c}\cdot\frac{c-a}{c+a}$. –ë–µ–∑ –æ–±–º–µ–∂–µ–Ω–Ω—è –∑–∞–≥–∞–ª—å–Ω–æ—Å—Ç—ñ –±—É–¥–µ–º–æ –≤–≤–∞–∂–∞—Ç–∏, —â–æ $a \geq b \geq c$, –ø–æ–∑–Ω–∞—á–∏–º–æ $b = c + \alpha$, $a = c + \alpha + \beta$. –î–ª—è —ñ—Å–Ω—É–≤–∞–Ω–Ω—è —Ç–∞–∫–æ–≥–æ —Ç—Ä–∏–∫—É—Ç–Ω–∏–∫–∞ –Ω–µ–æ–±—Ö—ñ–¥–Ω–æ –π –¥–æ—Å—Ç–∞—Ç–Ω—å–æ, —â–æ–± –≤–∏–∫–æ–Ω—É–≤–∞–ª–∞—Å—å —É–º–æ–≤–∞ $\beta < c$. –¢–æ–¥—ñ –∑–∞–¥–∞–Ω–∞ –≤ —É–º–æ–≤—ñ –Ω–µ—Ä—ñ–≤–Ω—ñ—Å—Ç—å –Ω–∞–±—É–≤–∞—î —Ç–∞–∫–æ–≥–æ –≤–∏–≥–ª—è–¥—É: $X = \frac{\alpha\beta(\alpha+\beta)}{(2c+\alpha)(2c+\alpha+\beta)(2c+2\alpha+\beta)} < k$. –û—Å–∫—ñ–ª—å–∫–∏ $2c > 2\beta$, —Ç–æ $X < \frac{\alpha\beta(\alpha+\beta)}{(2\beta+\alpha)(\alpha+3\beta)(2\alpha+3\beta)} = Y$. –ó–Ω–∞–π–¥–µ–º–æ —Ç–∞–∫–µ —á–∏—Å–ª–æ $k_1$, –ø—Ä–∏ —è–∫–æ–º—É $\forall\alpha$, $\beta > 0$ –≤–∏–∫–æ–Ω—É—î—Ç—å—Å—è —É–º–æ–≤–∞ $Y \leq k_1$. –ü–µ—Ä–µ–ø–∏—à–µ–º–æ –≤–∏—Ä–∞–∑: $Y = \frac{2\frac{\alpha}{2}((\beta + \frac{\alpha}{2}) - \frac{\alpha}{2})((\beta+\frac{\alpha}{2}) + \frac{\alpha}{2})}{2(\beta+\frac{\alpha}{2})(3(\beta+\frac{\alpha}{2})+\frac{\alpha}{2})(3(\beta+\frac{\alpha}{2})+\frac{\alpha}{2})} \leq k_1$. –ü–æ–∫–ª–∞–¥–µ–º–æ $x=\beta+\frac{\alpha}{2}$, $y=\frac{\alpha}{2}$. –¢–æ–¥—ñ –æ—Å—Ç–∞–Ω–Ω—è –Ω–µ—Ä—ñ–≤–Ω—ñ—Å—Ç—å –Ω–∞–±—É–¥–µ —Ç–∞–∫–æ–≥–æ –≤–∏–≥–ª—è–¥—É $\frac{y(x+y)(x-y)}{x(3x+y)(3x-y)} \le k_1 \iff \frac{1}{k_1} \le \frac{\left(9(\frac{x}{y})^2 - 1\right)\frac{x}{y}}{(\frac{x}{y})^2-1}$, –¥–∞–ª—ñ –∑–Ω–æ–≤—É –ø–æ–∑–Ω–∞—á–∏–º–æ $t = \frac{x}{y} = \frac{2\beta+\alpha}{\alpha} > 1$, —Ç–æ–¥—ñ $\frac{1}{k_1} \le \frac{(9t^2-1)t}{t^2-1}$. –ó–Ω–∞–π–¥–µ–º–æ –Ω–∞–π–º–µ–Ω—à–µ –∑–Ω–∞—á–µ–Ω–Ω—è —Ñ—É–Ω–∫—Ü—ñ—ó $f(t) = \frac{(9t^2-1)t}{t^2-1}$ –ø—Ä–∏ $t > 1$. –û—Å–∫—ñ–ª—å–∫–∏ $f'(t) = \frac{9t^4 - 26t^2 + 1}{(t^2-1)^2}$, —Ç–æ –º–æ–∂–µ–º–æ –∑–Ω–∞–π—Ç–∏ –µ–∫—Å—Ç—Ä–µ–º–∞–ª—å–Ω—ñ —Ç–æ—á–∫–∏: $t^2 = \frac{13 \pm 4\sqrt{10}}{9}$, —É–º–æ–≤—ñ $t > 1$ –∑–∞–¥–æ–≤–æ–ª—å–Ω—è—î –ª–∏—à–µ —Ç–æ—á–∫–∞ $t_0 = \sqrt{\frac{13+4\sqrt{10}}{9}} = \frac{\sqrt{5}+2\sqrt{2}}{3}$, –ª–µ–≥–∫–æ –ø–µ—Ä–µ–∫–æ–Ω–∞—Ç–∏—Å—å, —â–æ —Ü–µ —î —à—É–∫–∞–Ω–∞ —Ç–æ—á–∫–∞ –º—ñ–Ω—ñ–º—É–º—É. –ü—Ä–∏ —Ü—å–æ–º—É $f(t_0) = f\left(\frac{\sqrt{5}+2\sqrt{2}}{3}\right) = 8\sqrt{2}+5\sqrt{5}$, –æ—Ç–∂–µ $k_1 \ge \frac{1}{8\sqrt{2}+5\sqrt{5}} =\frac{8\sqrt{2}-5\sqrt{5}}{3}$ —ñ $k_1=\frac{8\sqrt{2}-5\sqrt{5}}{3}$.

–î–æ–≤–µ–¥–µ–º–æ, —â–æ $k = k_1$. –û—á–µ–≤–∏–¥–Ω–∏–º —î —Ç–æ–π —Ñ–∞–∫—Ç, —â–æ $k \geq k_1$. –ü—Ä–∏–ø—É—Å—Ç–∏–º–æ, —â–æ $k > k_1$. –í –ø–æ–ø–µ—Ä–µ–¥–Ω—ñ—Ö –ø–æ–∑–Ω–∞—á–µ–Ω–Ω—è—Ö: $t_0 = \frac{x_0}{y_0} = \frac{\beta_0 + \frac{1}{2}\alpha_0}{\frac{1}{2}\alpha_0} = \frac{2\beta_0}{\alpha_0}+1 \implies \beta_0 = \frac{(t_0-1)\alpha_0}{2}$. –í–∏–±–µ—Ä–µ–º–æ $\alpha$ —ñ $\beta$, —Ç–∞–∫–∏–º —á–∏–Ω–æ–º, —â–æ–± –≤–∏–∫–æ–Ω—É–≤–∞–ª–æ—Å—è —Å–ø—ñ–≤–≤—ñ–¥–Ω–æ—à–µ–Ω–Ω—è $\beta_0 = \frac{(t_0-1)\alpha_0}{2}$. –†–æ–∑–≥–ª—è–Ω–µ–º–æ —Ñ—É–Ω–∫—Ü—ñ—é $X(c) = \frac{(2\alpha\beta)(\alpha+\beta)}{(2c+\alpha)(2c+\alpha+\beta)(2c+2\alpha+\beta)}$. –í–æ–Ω–∞ –Ω–µ–ø–µ—Ä–µ—Ä–≤–Ω–∞ —É —Ç–æ—á—Ü—ñ $c = \beta$ —ñ $X(\beta) = f(t_0) = 8\sqrt{2}+5\sqrt{5}$, —Ç–æ–º—É $\forall\epsilon > 0 \exists\delta > 0: \forall x\in(\beta - \delta, \beta + \delta) \implies |X(x) - X(\beta)| < \epsilon$. –ü–æ–∫–ª–∞–≤—à–∏ $\epsilon = \frac{1}{2}(k_1 - k)$ —ñ –≤–∑—è–≤—à–∏ –¥–µ—è–∫–µ $c$ –∑ —ñ–Ω—Ç–µ—Ä–≤–∞–ª—É $(\beta, \beta + \delta)$ –æ—Ç—Ä–∏–º–∞—î–º–æ –æ—Ç—Ä–∏–º–∞—î–º–æ –¥–µ—è–∫–∏–π —Ç—Ä–∏–∫—É—Ç–Ω–∏–∫ –∑—ñ —Å—Ç–æ—Ä–æ–Ω–∞–º–∏ $c$, $b=c+\alpha$, $a=c+\alpha+\beta$, –¥–ª—è —è–∫–æ–≥–æ $k_1 > X(c)>k$. –û—Ç—Ä–∏–º–∞–Ω–µ –ø—Ä–æ—Ç–∏—Ä—ñ—á—á—è –¥–æ–≤–æ–¥–∏—Ç—å, —â–æ $k=k_1=\frac{8\sqrt{2}-5\sqrt{5}}{3}$.
</details></details>

	–Ø–∫—â–æ —Ç–∞–∫–∏—Ö –º–Ω–æ–∂–∏–Ω $X$ –Ω–µ —ñ—Å–Ω—É—î, —Ç–æ –∑–∞—Ä–∞–∑ –ø–æ—Ä–∞—Ö—É—î–º–æ, —Å–∫—ñ–ª—å–∫–∏ –º–Ω–æ–∂–∏–Ω –º—ñ—Å—Ç–∏—Ç—å —É —Å–æ–±—ñ $D$ —á–∏ $CD$. –£—Å—å–æ–≥–æ $2^{|CD|}$ –º–Ω–æ–∂–∏–Ω –º—ñ—Å—Ç—è—Ç—å $D$, —Ç–∞ $2^{|D|}$ –º–Ω–æ–∂–∏–Ω –º—ñ—Å—Ç—è—Ç—å $CD$. –ê–ª–µ —É—Å—è –º–Ω–æ–∂–∏–Ω–∞ —Ä–∞—Ö—É–≤–∞–ª–∞—Å—è –¥–≤—ñ—á—ñ, —Ç–∞–∫–∏–º —á–∏–Ω–æ–º —É—Å—å–æ–≥–æ —Ç–∞–∫–∏–º –º–Ω–æ–∂–∏–Ω $2^{|CD|} + 2^{|D|} - 1$. –¢–∞–∫ —Å–∞–º–æ –ø—ñ–¥—Ä–∞—Ö–æ–≤–∞–Ω–æ –º–Ω–æ–∂–∏–Ω–∏ $D$ —Ç–∞ $CD$, —è–∫—ñ –Ω–µ –Ω–∞–ª–µ–∂–∞—Ç—å $F$. –ó–≤—ñ–¥—Å–∏ $2^{n-1}-1 = |F| \ge 2^{|CD|} + 2^{|D|} - 3$. –ù–µ—Ö–∞–π $\max\{|D|,|CD|\}=t$, —Ç–æ $2^{n-1} -1 \le 2^t + 2^t -3=2^{t+1}-3$. –¶–µ –ø–æ–∫–∞–∑—É—î, —â–æ $t \ge n-1$, –æ—Å–∫—ñ–ª—å–∫–∏ $D$ —Ç–∞ $CD$ –Ω–µ –ø–æ—Ä–æ–∂–Ω—ñ, —Ç–æ $t=n-1$. –ë–µ–∑ –æ–±–º–µ–∂–µ–Ω—å –∑–∞–≥–∞–ª—å–Ω–æ—Å—Ç—ñ –≤–≤–∞–∂–∞—î–º–æ, —â–æ $|D|=1$, $|CD|=n-1$. –ê–ª–µ —Ç–æ–¥—ñ –∫–æ–∂–Ω–∞ –º–Ω–æ–∂–∏–Ω–∞, —â–æ –º—ñ—Å—Ç–∏—Ç—å $CD$, —Ü–µ —î –∞–±–æ —Å–∞–º–∞ –º–Ω–æ–∂–∏–Ω–∞ $CD$, –∞–±–æ –ø–æ–≤–Ω–∞ –º–Ω–æ–∂–∏–Ω–∞. –û—Å–∫—ñ–ª—å–∫–∏ $CD$ –Ω–µ –º—ñ—Å—Ç–∏—Ç—å—Å—è –≤ $F$, —Ç–æ –ª–∏—à–µ –ø–æ–≤–Ω–∞ –º–Ω–æ–∂–∏–Ω–∞ –º–æ–∂–µ –º—ñ—Å—Ç–∏—Ç–∏ $CD$, –∞–ª–µ —Ü—è –º–Ω–æ–∂–∏–Ω–∞ —Ç–∞–∫ —Å–∞–º–æ –º—ñ—Å—Ç–∏—Ç—å —ñ $D$. –¢–æ–±—Ç–æ –∫–æ–∂–Ω–∞ –º–Ω–æ–∂–∏–Ω–∞ –∑ $F$ –º—ñ—Å—Ç–∏—Ç—å $C$, —â–æ –π –æ–∑–Ω–∞—á–∞—î, —â–æ –ø–µ—Ä–µ—Ç–∏–Ω —É—Å—ñ—Ö –º–Ω–æ–∂–∏–Ω –∑ $F$ –Ω–µ–ø–æ—Ä–æ–∂–Ω—ñ–π.

## 4
–ù–µ—Ö–∞–π $n$ ‚Äì –¥–µ—è–∫–µ –Ω–∞—Ç—É—Ä–∞–ª—å–Ω–µ —á–∏—Å–ª–æ. –ó–Ω–∞–π—Ç–∏ –≤—Å—ñ —Ñ—É–Ω–∫—Ü—ñ—ó $f: \mathbb{R}^+ \to \mathbb{R}$ (—Ç–æ–±—Ç–æ —Ñ—É–Ω–∫—Ü—ñ—ó, –≤–∏–∑–Ω–∞—á–µ–Ω—ñ –Ω–∞ –º–Ω–æ–∂–∏–Ω—ñ —É—Å—ñ—Ö –¥–æ–¥–∞—Ç–Ω–∏—Ö –¥—ñ–π—Å–Ω–∏—Ö —á–∏—Å–µ–ª –∑ –¥—ñ–π—Å–Ω–∏–º–∏ –∑–Ω–∞—á–µ–Ω–Ω—è–º–∏), –¥–ª—è —è–∫–∏—Ö –≤–∏–∫–æ–Ω—É—î—Ç—å—Å—è —Ä—ñ–≤–Ω—ñ—Å—Ç—å $f(x^{n+1}) + f(y^{n+1}) = x^n f(x) + y^n f(y)$ –¥–ª—è –±—É–¥—å-—è–∫–∏—Ö –¥–æ–¥–∞—Ç–Ω–∏—Ö –¥—ñ–π—Å–Ω–∏—Ö —á–∏—Å–µ–ª $x$, $y$.
<details><summary>–í—ñ–¥–ø–æ–≤—ñ–¥—å</summary>

$f(x) = ax$, $a$ ‚Äì –¥–æ–≤—ñ–ª—å–Ω–µ –¥—ñ–π—Å–Ω–µ —á–∏—Å–ª–æ.
<details><summary>–†–æ–∑–≤‚Äô—è–∑–∞–Ω–Ω—è</summary>

–ü—ñ–¥—Å—Ç–∞–≤–∏–º–æ $x = y$, –æ—Ç—Ä–∏–º–∞—î–º–æ $f(2x^{n+1}) = 2x^n f(x)$, –∞ —Ç–æ–º—É $2 f(x^{n+1} + y^{n+1}) = 2x^n f(x) + 2y^n f(y) = f(2x^{n+1}) + f(2y^{n+1})$. –ó–∞–º—ñ–Ω—é—é—á–∏ $x^{n+1}$ —ñ $y^{n+1}$ –Ω–∞ $a$ —ñ $b$ –≤—ñ–¥–ø–æ–≤—ñ–¥–Ω–æ, –æ—Ç—Ä–∏–º–∞—î–º–æ —Ä—ñ–≤–Ω—ñ—Å—Ç—å
```math
2 f(a + b) = f(2a) + f(2b). \tag{1}
```
–†–æ–∑–≥–ª—è–Ω–µ–º–æ –≤–∏—Ä–∞–∑ $4 f(a + b + c)$. –ó –æ–¥–Ω–æ–≥–æ –±–æ–∫—É, –≤—ñ–Ω –¥–æ—Ä—ñ–≤–Ω—é—î $2 f(2a) + 2 f(2b + 2c) = 2 f(2a) + f(4b) + f(4c)$, –∑ —ñ–Ω—à–æ–≥–æ ‚Äì $f(4a) + f(4b) + 2 f(2c)$, –∑–≤—ñ–¥–∫–∏ $f(4a) ‚àí 2 f(2a) = f(4c) ‚àí 2 f(2c)$. –û—Å–∫—ñ–ª—å–∫–∏ $f(2x^{n+1}) = 2x^n f(x)$, —Ç–æ $f(2) = 2 f(1)$ —ñ –ø—ñ–¥—Å—Ç–∞–≤–ª—è—é—á–∏ —É –ø–æ–ø–µ—Ä–µ–¥–Ω—é —Ä—ñ–≤–Ω—ñ—Å—Ç—å $x = \frac{a}{2}$, $c = \frac{1}{2}$, –º–∞—Ç–∏–º–µ–º–æ $f(2x) ‚àí 2 f(x) = f(2) ‚àí 2 f(1) = 0$. –¢–æ–º—É, –∑ —É—Ä–∞—Ö—É–≤–∞–Ω–Ω—è–º (1), —Ñ—É–Ω–∫—Ü—ñ—è $f$ ‚Äì –∞–¥–∏—Ç–∏–≤–Ω–∞. –¢–∞–∫–∏–º —á–∏–Ω–æ–º, –º–∞—î–º–æ –¥–≤—ñ —É–º–æ–≤–∏ –Ω–∞ $f$: $f(x + y) = f(x) + f(y)$ —ñ $f(x^{n+1}) = x^n f(x)$.

–ó –∞–¥–∏—Ç–∏–≤–Ω–æ—Å—Ç—ñ $f$ –≤–∏–ø–ª–∏–≤–∞—î, —â–æ $f(kx) = kf(x)$ –¥–ª—è $k \in \mathbb{N}$. –ù–µ—Ö–∞–π $x = t + k$, –¥–µ $k$ ‚Äì –Ω–∞—Ç—É—Ä–∞–ª—å–Ω–µ. –¢–æ–¥—ñ $f((t + k)^{n+1}) = (t+k)^n f(t+k)$, –∞ –æ—Ç–∂–µ, $\sum_{i=0}^{n+1} f\left(C^{n+1}_i t^i k^{n+1-i}\right) = (t+k)^n(f(t)+f(k))$, $\sum_{i=0}^{n+1} C_{n+1}^i k^i f(t^{n+1-i}) - (t+k)^n(f(t)+kf(1))=0$. –ó–∞—Ñ—ñ–∫—Å—É—î–º–æ $t$ —ñ —Ä–æ–∑–≥–ª—è–Ω–µ–º–æ –æ—Å—Ç–∞–Ω–Ω—ñ–π –≤–∏—Ä–∞–∑ —è–∫ –º–Ω–æ–≥–æ—á–ª–µ–Ω –≤—ñ–¥–Ω–æ—Å–Ω–æ $k$. –í—ñ–Ω –¥–æ—Ä—ñ–≤–Ω—é—î 0 –≤ —É—Å—ñ—Ö –Ω–∞—Ç—É—Ä–∞–ª—å–Ω–∏—Ö —Ç–æ—á–∫–∞—Ö, –∞ —Ç–æ–º—É –≤—ñ–Ω —Ç–æ—Ç–æ–∂–Ω–∏–π 0. –ó–Ω–∞–π–¥–µ–º–æ –∫–æ–µ—Ñ—ñ—Ü—ñ—î–Ω—Ç –ø—Ä–∏ $k^n$. –ó –æ–¥–Ω–æ–≥–æ –±–æ–∫—É, –≤—ñ–Ω —Ä—ñ–≤–Ω–∏–π $C_{n+1}^n f(t) -f(t)- C_n^1 f(1)t$, –∑ —ñ–Ω—à–æ–≥–æ –≤—ñ–Ω ‚Äì 0, —Ç–æ–º—É $f(t) = f(1)t$. –†–æ–∑–≥–ª—è–¥–∞—é—á–∏ –¥–æ–≤—ñ–ª—å–Ω–µ $t$, –æ—Ç—Ä–∏–º–∞—î–º–æ $f(t) = at$, –¥–µ $a = f(1) \in \mathbb{R}$. –ë–µ–∑–ø–æ—Å–µ—Ä–µ–¥–Ω—å–æ—é –ø–µ—Ä–µ–≤—ñ—Ä–∫–æ—é –ø–µ—Ä–µ–∫–æ–Ω—É—î–º–æ—Å—å, —â–æ $f(x) = ax$ –∑–∞–¥–æ–≤–æ–ª—å–Ω—è—î —É–º–æ–≤—É –∑–∞–¥–∞—á—ñ.
</details></details>

## 5
–ù–µ—Ö–∞–π $A$, $B$, $C$, $D$ —Ç–∞ $E$ ‚Äì –ø–æ—Å–ª—ñ–¥–æ–≤–Ω—ñ —Ç–æ—á–∫–∏ –Ω–∞ –∫–æ–ª—ñ –∑ —Ü–µ–Ω—Ç—Ä–æ–º –≤ —Ç–æ—á—Ü—ñ $O$, —Ç–∞–∫—ñ, —â–æ $AC = BD = CE = DO$. –ù–µ—Ö–∞–π —Ç–æ—á–∫–∏ $H_1$, $H_2$ —Ç–∞ $H_3$ ‚Äì —Ü–µ —Ç–æ—á–∫–∏ –ø–µ—Ä–µ—Ç–∏–Ω—É –≤–∏—Å–æ—Ç —Ç—Ä–∏–∫—É—Ç–Ω–∏–∫—ñ–≤ $ACD$, $BCD$ —Ç–∞ $BCE$ –≤—ñ–¥–ø–æ–≤—ñ–¥–Ω–æ. –î–æ–≤–µ—Å—Ç–∏, —â–æ —Ç—Ä–∏–∫—É—Ç–Ω–∏–∫ $H_1H_2H_3$ ‚Äì –ø—Ä—è–º–æ–∫—É—Ç–Ω–∏–π.
<details><summary>–†–æ–∑–≤‚Äô—è–∑–∞–Ω–Ω—è 1</summary>

–¢–∞–∫ —è–∫ —Ç–æ—á–∫–∞ $H_1$ ‚Äì –æ—Ä—Ç–æ—Ü–µ–Ω—Ç—Ä —Ç—Ä–∏–∫—É—Ç–Ω–∏–∫–∞ $ACD$, –∞ $O$ ‚Äì —Ü–µ–Ω—Ç—Ä –æ–ø–∏—Å–∞–Ω–æ–≥–æ –∫–æ–ª–∞ —Ü—å–æ–≥–æ —Ç—Ä–∏–∫—É—Ç–Ω–∏–∫–∞, —Ç–æ $\overrightarrow{OH_1} = \overrightarrow{OA} + \overrightarrow{OC} + \overrightarrow{OD}$. –ê–Ω–∞–ª–æ–≥—ñ—á–Ω–æ, $\overrightarrow{OH_2} = \overrightarrow{OB} + \overrightarrow{OC} + \overrightarrow{OD}$ —Ç–∞ $\overrightarrow{OH_3} = \overrightarrow{OB} + \overrightarrow{OC} + \overrightarrow{OE}$. –û—Ç–∂–µ, $\overrightarrow{H_1H_2} = \overrightarrow{OH_2} ‚àí \overrightarrow{OH_1} = \overrightarrow{OB} ‚àí \overrightarrow{OA}$ —Ç–∞ $\overrightarrow{H_2H_3} = \overrightarrow{OH_3} ‚àí \overrightarrow{OH_2} = \overrightarrow{OE} ‚àí \overrightarrow{OD}$. –°–∫–∞–ª—è—Ä–Ω–∏–π –¥–æ–±—É—Ç–æ–∫ —Ü–∏—Ö –≤–µ–∫—Ç–æ—Ä—ñ–≤ –¥–æ—Ä—ñ–≤–Ω—é—î
```math
\overrightarrow{H_1H_2} \cdot \overrightarrow{H_2H_3} = (\overrightarrow{OB} ‚àí \overrightarrow{OA})(\overrightarrow{OE} ‚àí \overrightarrow{OD}) = \overrightarrow{OB} \cdot \overrightarrow{OE} ‚àí \overrightarrow{OB} \cdot \overrightarrow{OD} ‚àí \overrightarrow{OA} \cdot \overrightarrow{OE} + \overrightarrow{OA} \cdot \overrightarrow{OD} = |OB|\cdot|OE|\cos\angle BOE - |OB|\cdot|OD|\cos\angle BOD - |OA|\cdot|OE|\cos\angle AOE + |OA|\cdot|OD|\cos\angle AOD = |OA|^2 (\cos \angle BOE ‚àí \cos \angle BOD ‚àí \cos \angle AOE + \cos \angle AOD).
```
–¢—Ä–∏–∫—É—Ç–Ω–∏–∫–∏ $AOC$ —Ç–∞ $COE$ —Ä—ñ–≤–Ω–æ—Å—Ç–æ—Ä–æ–Ω–Ω—ñ, —Ç–æ–º—É $\angle AOE =120^\circ$. –¢—Ä–∏–∫—É—Ç–Ω–∏–∫ $BOD$ —Ç–∞–∫–æ–∂ —Ä—ñ–≤–Ω–æ—Å—Ç–æ—Ä–æ–Ω–Ω—ñ–π, —Ç–æ–º—É $\angle BOD = 60^\circ$. –û—Ç–∂–µ,
```math
\overrightarrow{H_1H_2} \cdot \overrightarrow{H_2H_3} = |OA|^2 (\cos \angle BOE ‚àí \cos 60^\circ ‚àí \cos 120^\circ + \cos \angle AOD) = |OA|^2 (\cos \angle BOE + \cos \angle AOD) = 2|OA|^2 \cos \left(\frac{\angle BOE + \angle AOD}{2}\right) \cos \left(\frac{\angle BOE ‚àí \angle AOD}{2}\right).
```
